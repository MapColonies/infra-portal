# Grafana

You can access Libot's Grafana at [https://grafana.mapcolonies.net](https://grafana.mapcolonies.net).

Grafana is our central visualization and observability platform. It allows you to query, visualize, and analyze metrics, logs, and traces 
from multiple data sources (like Prometheus, Loki, and Tempo) in a single unified interface.

![grafana dashboard](/img/monitoring/grafana/grafana-dashboard.png)

## Authentication

Access to Grafana is secured and streamlined using **GitHub SSO**. You can log in using your standard GitHub credentials.

## Installed Plugins

We have extended Grafana's core capabilities with the following plugins:

* **[`grafana-github-datasource`](https://grafana.com/grafana/plugins/grafana-github-datasource/)**: 
Allows us to pull data directly from GitHub into our dashboards.
* **[`magnesium-wordcloud-panel`](https://grafana.com/grafana/plugins/magnesium-wordcloud-panel/)**: 
Provides custom visual panels for generating word clouds from log or metric data.
* **[`redis-app`](https://grafana.com/grafana/plugins/redis-app/)**: 
A comprehensive application plugin that includes ready-to-use dashboards and a dedicated UI for querying Redis directly.

## AI Integration

We have integrated an LLM directly into Grafana via the Model Context Protocol (MCP) using a custom service account.  
This allows you to interact with Grafana using natural language to generate or modify dashboards. 

To ensure the safety of our core production dashboards, we are currently operating in a **grace period**.  
During this time, the AI is strictly restricted to creating and modifying resources *only* within the designated **`AI`** folder. 

For instructions on how to set up this connection in your AI client, please see the **[Adding Grafana MCP to an LLM Guide](../../guides/monitoring/grafana.mdx#adding-grafana-mcp-to-an-llm)**.

## Infrastructure as Code ([Upcoming](https://mapcolonies.atlassian.net/browse/MAPCO-10563))

Currently, alerts and dashboards are managed within the UI.  
However, we are in the process of migrating to a fully automated GitOps approach using Grafana sidecar ConfigMaps.  
Once complete, all dashboards and alerts will be provisioned automatically from our infrastructure repositories.
