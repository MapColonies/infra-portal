# 📊 Grafana

You can access Libot's Grafana at [https://grafana.mapcolonies.net](https://grafana.mapcolonies.net).

**Grafana** is our ultimate, single-pane-of-glass observability platform. It allows you to query, visualize, and analyze metrics, logs, and traces from *all* our data sources (like Prometheus, Loki, and Tempo) in one beautiful interface!

![grafana dashboard](/img/monitoring/grafana/grafana-dashboard.png)

## 🔐 Authentication

Logging in is effortless. Access is secured using **GitHub SSO**, so you can jump right in using your standard GitHub credentials!

## 🧩 Supercharged Plugins

We have turbocharged Grafana's core capabilities with these powerful plugins:

* 🐙 **[`grafana-github-datasource`](https://grafana.com/grafana/plugins/grafana-github-datasource/):** Pulls data directly from GitHub into our dashboards.
* ☁️ **[`magnesium-wordcloud-panel`](https://grafana.com/grafana/plugins/magnesium-wordcloud-panel/):** Generates sleek, custom word clouds from log and metric data!
* ⚡ **[`redis-app`](https://grafana.com/grafana/plugins/redis-app/):** The ultimate Redis plugin. It includes ready-to-use dashboards and a dedicated UI for querying Redis natively.

## 🤖 AI Integration

We've wired a powerful LLM directly into Grafana using the Model Context Protocol (MCP) and a custom service account! 
This allows you to chat with Grafana in natural language to generate or modify dashboards. ✨

:::info 🛡️ Grace Period Restictions
To keep our production dashboards completely safe, we are currently operating in a **grace period**. 
The AI is strictly sandboxed—it can only create and modify resources inside the designated **`AI`** folder. 
:::

Want to hook up your local AI client? Check out the **[Adding Grafana MCP to an LLM Guide](../../guides/monitoring/grafana.mdx#adding-grafana-mcp-to-an-llm)**!

## 🏗️ Infrastructure as Code ([Upcoming](https://mapcolonies.atlassian.net/browse/MAPCO-10563))

Right now, alerts and dashboards are built and managed manually within the UI. 
**But not for long!** We are migrating to a fully automated GitOps approach using Grafana sidecar ConfigMaps. Soon, every dashboard and alert will be provisioned automatically straight from our infra repos! 🚀
