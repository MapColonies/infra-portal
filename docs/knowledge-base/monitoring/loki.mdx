You can access Loki data by navigating to the **Explore** tab within [Grafana](https://grafana.mapcolonies.net).

Loki is our horizontally scalable, highly available, multi-tenant log aggregation system. Unlike other logging systems, Loki does not index the contents of the logs, but rather a set of labels for each log stream. This makes it incredibly lightweight and cost-effective.

## Architecture & Storage

* **Ingestion:** Loki receives log streams primarily from our **Grafana Alloy** deployment, which scrapes pod logs across the cluster.
* **Storage:** We utilize **S3** as the backend object storage for Loki. This allows us to store massive volumes of log data securely and cost-effectively without managing complex local storage volumes.
* **Usage Focus:** Currently, Loki is used strictly for log search, exploration, and troubleshooting via Grafana. We are not utilizing Loki Ruler for log-based alerting at this time.
