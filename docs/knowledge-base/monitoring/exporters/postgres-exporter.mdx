The Postgres Exporter provides critical visibility into the health and performance of our central PostgreSQL database.

## Monitoring Scope

We maintain a single, centrally managed PostgreSQL instance for the environment. 
The Postgres Exporter is configured globally by the infrastructure team to connect to this database and 
expose standard performance metrics (such as active connections, transaction rates, and buffer behavior) to Prometheus.

Because we rely entirely on standard, built-in metrics, we maintain a consistent baseline for database observability without the overhead of maintaining custom queries.
