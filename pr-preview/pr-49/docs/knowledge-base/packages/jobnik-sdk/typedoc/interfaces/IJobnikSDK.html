IJobnikSDK

@map-colonies/jobnik-sdk

Interface: IJobnikSDK<JobTypes, StageTypes>

Defined in: types/sdk.ts:40

Main interface for the Jobnik SDK providing access to job management clients. Central API for creating jobs, consuming tasks, and managing worker processes.

Example​

interface MyJobTypes {
  'image-processing': {
    userMetadata: { userId: string };
    data: { imageUrl: string };
  };
}

interface MyStageTypes {
  'resize': {
    userMetadata: { quality: number };
    data: { width: number; height: number };
    task: {
      userMetadata: { batchId: string };
      data: { sourceUrl: string };
    };
  };
}

const sdk: IJobnikSDK<MyJobTypes, MyStageTypes> = new JobnikSDK({
  baseUrl: 'https://api.example.com'
});

createWorker​

createWorker: <JobType, StageType>(stageType, taskHandler, options) => IWorker

Defined in: types/sdk.ts:83

Creates a worker instance to continuously process tasks of a specific stage type.

Type Parameters​

JobType​

JobType extends string | string & object = string

StageType​

StageType extends string | string & object = string

The stage type this worker will process

Parameters​

stageType​

StageType

Stage type name to process tasks for

taskHandler​

TaskHandler<JobTypes, StageTypes, JobType, StageType>

Function to handle individual task processing

options​

WorkerOptions

Optional worker configuration (concurrency, intervals, circuit breakers)

Returns​

IWorker

Worker instance for continuous task processing

Example​

const worker = sdk.createWorker(
  async (task, context) => {
    context.logger.info('Processing task', { taskId: task.id });
    // Process task data...
  },
  'resize',
  { concurrency: 3, pullingInterval: 5000 }
);
